# Agentic AI System for Resume–Job Matching and Cover Letter Generation using LangGraph also known as ai cover letter agent.

## 🧠 Overview

This project is a memory-enabled agentic AI system that generates personalized cover letters by analyzing resumes and job descriptions.

It is built using **LangGraph**, **LangChain**, and **Groq LLMs**, and applies structured multi-step reasoning with checkpoint-based memory for stateful execution.

---

## 🎯 Objective

To automate and improve cover letter generation by:
- Structuring resume and job description data
- Matching skills with job requirements
- Generating context-aware and personalized cover letters

---

## 🚀 Features

-  Resume parsing into structured format  
-  Job description parsing and extraction  
-  Skill matching using MatchMap  
-  AI-generated tailored cover letters  
-  Optional feedback-based refinement loop  
-  State memory using LangGraph checkpoints  
-  Agentic multi-step workflow  

---

## 🛠️ Tech Stack

- Python  
- LangChain  
- LangGraph  
- Groq API (LLMs)  

---

## 🧠 Memory System

The system uses **LangGraph checkpoint-based memory** to preserve workflow state across steps such as parsing, matching, and generation.
